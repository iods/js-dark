### Installation

##### Using Node

```sh
$ yarn install
```

```sh
$ yarn serve
```

##### Using Docker

```sh
# uncomment - '8080:3000'
# uncomment - ./Library/developer-react:/usr/share/nginx/html:delegated
$ docker-compose up
```
