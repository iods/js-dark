### What is Jest?

Jest is a testing Framework written by Facebook.

 1. Almost zero configuration required.
 2. Runs in parallel
 3. Runs in isolation 
 4. Powerful mocking support
 5. Built-in support for coverage out of the box
 6. Supports snapshot testing
 7. Has a rich API
