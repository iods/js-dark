Dom Process (How to Target Elements through JS)

  • Target Document

  • Use A Selector to Target

      getElementById :

    • can target id by using .id Notation
    • can target className by using .className Notation
    • can target getAttribute by using .getAttribute('attribute Name ex(Id)') Notation
    • can target setAttribute by using .setAttribute('attribute Name','what you change ') Notation
    • we can give access to this by a varible like cosnt title = document.getElementById Now we can access all the attribute through variabel title.
    • Accessing Attributes Example title.style.padding = '5px' As same as u can easily maipulate style by JS

    • innerText => Displays the content on window.(Here window means your Browser Tab).
      innerContent => It also displays the content on window also shows the hidden Content too
      innerHTML => It also displays the content but also shows the tags that are in the Element.
      Example :
      let container = document.getElementById('title')
      container.innerText => Output :'DOM'
      container.textContent => Output : 'DOM test text'
      container.innerHTML => Output : 'DOM "span style="display: none;" test text /span"'

  • getElementsByClassName
      Same As getElementsById
      var elements = document.getElementsByClassName('exampleClass');
  • getElementsByTagName
      Same As getElementsById
      var elements = document.getElementsByTagName('p');
  • querySelector
      Same As getElementsById
      var elements = document.querySelector('.exampleClass' / '#expampleId');
  • querySelectorAll
      Same As getElementsById
      var elements = document.querySelectorAll('.exampleClass'/'#example id');
  • getElementsByName
      Same As getElementsById
      var elements = document.getElementsByName('exampleName');
